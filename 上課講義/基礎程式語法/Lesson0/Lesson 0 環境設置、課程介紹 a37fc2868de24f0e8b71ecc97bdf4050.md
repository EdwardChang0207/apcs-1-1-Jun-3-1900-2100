### (3)選擇資料夾

在終端機中使用cd <path>即可進入目標資料夾，若使用cd ../ 可返回上一層

```python
cd <資料夾名稱>
```

### (4)執行檔案

使用python指令即可執行python檔案

```python
python example.py
```

## 三、常用快捷鍵

### (1)複製、貼上

複製: Ctrl + C

貼上: Ctrl + V

### (2)選取

一般選取: Shift + 方向鍵

選取多個重複項目: Ctrl + D

### (3)還原

Ctrl + Z

### (4)移動

Alt + 方向鍵

### (5)註解

Ctrl + /
